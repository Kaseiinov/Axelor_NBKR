# Модуль: Курсы валют НБКР — Axelor ERP


## Требования
- Java 21
- PostgreSQL 14+
- Gradle 8+

---

## Настройка базы данных
```sql
CREATE DATABASE axelor_erp;
CREATE USER axelor WITH PASSWORD 'axelor';
GRANT ALL PRIVILEGES ON DATABASE axelor_erp TO axelor;
```

---

## Запуск проекта
```bash
export JAVA_HOME=/Library/Java/JavaVirtualMachines/temurin-21.jdk/Contents/Home
./gradlew clean build run
```
Открыть в браузере: http://localhost:8080/axelor-demo
Логин: **admin**
Пароль: **admin**

---

## Что проверить

### 1. Меню модуля
Левое меню → **Курсы валют**
Должны быть два пункта:
- Список курсов
- Обновить курсы вручную

### 2. Ручная загрузка
Курсы валют → Обновить курсы вручную → нажать кнопку. Ожидаемый результат: уведомление "Курсы валют успешно обновлены!"

### 3. Данные сохранились
Курсы валют → Список курсов. Ожидаемый результат: таблица с валютами (USD, EUR, RUB и др.),
датой курса и значением по отношению к KGS

### 4. Нет дублирования
Нажать "Обновить курсы вручную" повторно. Ожидаемый результат: количество записей не изменилось,
данные за ту же дату обновились (не задублировались)

### 5. Автоматический планировщик
Administration → Job Management → All Schedules. Ожидаемый результат: запись "Загрузка курсов НБКР", com.axelor.contact.job.CurrencyRateJob, 
с CRON 0 0 9 * * ? и Active = включён

### 6. Планировщик динамический
Нажать на джоб → изменить CRON → сохранить. Ожидаемый результат: изменение применяется без перезапуска сервера
